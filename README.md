# Tesoon-jtjx
# Tesoon 主页访问指引

> 数据来源：`tesoon_bookids_1_300.csv` 与 `tesoon_bookids_1_300_courses.csv`

## 1. 主页入口

统一访问地址：

```text
https://m24.tesoon.com/tszz/jtjx/?bookId=书籍编号
```

其中 `bookId` 是最重要的参数。把书籍编号替换进去，就能打开对应主页。

### 示例

- `bookId=38` → `https://m24.tesoon.com/tszz/jtjx/?bookId=38`
- `bookId=49` → `https://m24.tesoon.com/tszz/jtjx/?bookId=49`
- `bookId=119` → `https://m24.tesoon.com/tszz/jtjx/?bookId=119`
- `bookId=192` → `https://m24.tesoon.com/tszz/jtjx/?bookId=192`

## 2. 使用方法

1. 打开主页链接。
2. 在链接里替换 `bookId` 为目标编号。
3. 页面返回 `ok=true` 且 `httpStatus=200` 时，说明该书籍可正常访问。
4. 如果返回 `ok=false`、`httpStatus=400`，并提示 **“图书信息不存在”**，说明该 `bookId` 无效或当前没有对应图书。
5. 进入书籍页后，可在 `视频资源 / 排行榜 / 我的收藏` 等标签中浏览内容。

可访问书籍主要集中在以下几类：

- **系列**：`解题觉醒`、`高中解题觉醒`、`高考解题觉醒`

## 5. 书籍页常见字段说明

| 字段 | 含义 |
|---|---|
| `bookId` | 书籍编号，用于拼接主页链接 |
| `ok` | 是否可访问 |
| `httpStatus` / `apiStatus` | 请求状态 |
| `message` | 返回信息 |
| `name` | 书籍名称 |
| `series` | 系列名 |
| `grade_term` | 年级或学期 |
| `subject_or_book` | 学科或教材信息 |
| `version` | 教材版本 |
| `need_buy` | 是否需要购买 |
| `is_need_active` | 是否需要激活 |
| `is_activated` | 当前激活状态 |
| `tabs` | 页面标签 |
| `categories` | 分类信息 |

## 6. 课程页字段说明

书籍页下方的课程数据主要包括：

| 字段 | 含义 |
|---|---|
| `courseId` | 课程编号 |
| `courseTitle` | 课程标题 |
| `courseDesc` | 课程简介 |
| `thumb` | 课程封面 |
| `videosTotal` | 视频数量 |
| `studiedTotal` | 学习人数 |
| `supportingResourcesTitle` | 配套资料标题 |
| `supportingResourcesTotal` | 配套资料数量 |

## 7. 推荐查看方式

建议按下面顺序使用：

- 先用 `bookId` 打开书籍主页
- 再查看该页下的课程标题
- 根据 `videosTotal` 判断课程体量
- 根据 `supportingResourcesTotal` 判断是否有配套资料
